# DeweyGraph File Organizer (Strict v2)

This version includes:
- archival worthiness filtering
- material role awareness
- notebook handling rules
- folder typology awareness
- explicit duplicate limitations

## Pre-classification step

Before classification, apply:

1. Archival Worthiness Filter
2. Folder Typology
3. Material Role assignment (if applicable)

## Duplicate rule

Duplicate detection is advisory unless supported by:
- file hashes
- timestamps
- content comparison

Do not assert canonical versions without evidence.

## Notebook rule

Classify notebooks by role in archive, not mixed content.

## Material role

Each item may have one role:
- source
- process
- reference
- output
- residue
- duplicate-candidate

## Folder awareness

Adjust behaviour based on folder type:
- intake → triage first
- project → preserve structure
- archive → classify deeply
- dump → organise before classify

## Core rule

Do not classify everything.
Only classify what is worth remembering.

## Everything else

Follows strict skill rules.
