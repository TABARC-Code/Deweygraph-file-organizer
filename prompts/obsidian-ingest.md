# Obsidian Ingest Prompt

Use **DeweyGraph File Organizer** in **CLASSIFY + OBSIDIAN_NOTE + GRAPH_MAP** mode.

Treat the folder as archival material.

For each meaningful item:
- assign one anchor code
- add 1–4 secondary domains
- add 1–4 lens tags
- create typed bi-directional relation links
- write a short decision note
- output an Obsidian markdown note

Also create a folder-level index note if the directory functions as a coherent archive or project collection.
