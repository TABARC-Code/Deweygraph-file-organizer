# Duplicate Audit Prompt

Use **DeweyGraph File Organizer** in **ANALYZE_STRUCTURE + DUPLICATE_REVIEW + ORGANIZE_PLAN** mode.

Review the folder for:
- exact or likely duplicates
- duplicate project folders
- old versions
- poor naming patterns
- active vs archive confusion

Recommend:
- which files to keep
- which to archive
- which to delete (only when clearly safe)
- folder consolidation where appropriate

Do not delete anything automatically.
Return a clear plan before any structural changes.
