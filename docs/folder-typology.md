# Folder Typology

Not all folders are the same.

Treating them as identical leads to bad organisation decisions.

## Types

### intake
Unsorted incoming material.

### project
Coherent working set of files.

### archive
Completed or stable material.

### reference
Supporting material only.

### export
Generated outputs.

### dump
Unstructured mix.

### mixed
Combination of the above.

## Rule

Classify folder type before reorganising.

## Why

Organisation depends on context.

A project folder should not be treated like Downloads.
